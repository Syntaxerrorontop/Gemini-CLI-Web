# Gemini AutoAgent v2

**Gemini AutoAgent** is an advanced, fully autonomous AI engineering suite designed to transform high-level requirements into fully functional software architectures. By orchestrating multiple specialized AI agents, the system builds, tests, and validates software in real-time within a secure local environment.

## 🚀 Key Features

*   **Autonomous Roadmap Orchestration:** Break down any software idea into a technical execution plan.
*   **Live Coding Terminal:** Watch the AI agent implement code, research the file system, and fix bugs in real-time.
*   **Customizable AI Core:** Modify the planning and worker prompts to fine-tune the agent's behavior for your specific needs.
*   **Project History & State Recovery:** Roll back to previous versions of your roadmap or restore archived development sessions.
*   **Full Privacy:** The system runs locally and communicates only with the Gemini API via the secure Gemini CLI.

## 🛠️ Tech Stack

*   **Frontend:** React (TypeScript) + Vite + TailwindCSS + Lucide Icons
*   **Backend:** Python (FastAPI) + WebSockets
*   **AI Engine:** Google Gemini (via `@google/gemini-cli`)
